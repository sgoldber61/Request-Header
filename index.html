Request Header Parser Microservice

This is an API project for FreeCodeCamp with the following functionality (mostly copied from the FreeCodeCamp site):

  1. I can get the IP address, language and operating system for my browser. This is returned in json format.

Example usage:

/whoami

Example output:

{ipaddress: "72.198.91.159", language: "en-US", software: "Macintosh; Intel Mac OS X 10_11_1"}
